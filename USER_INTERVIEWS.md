User Interviews: AI Spend Auditor Validation
Overview
I conducted three interviews with distinct user personas to validate the "AI Spend" problem. These conversations revealed a deep tension between technical efficiency (APIs), operational neglect (seat ghosts), and UI loyalty (the productivity tax).

Interview 1: The "Efficiency Hacker"
Role: Solo Full-Stack Developer
Current Stack: Claude Code (CLI/API-based usage)

Key Findings
Price Sensitivity: Proactively moved away from Cursor to save costs.

Technical Trade-off: Prioritizes higher usage limits via CLI over a polished GUI.

Validation: Confirms that highly technical users will churn from $20/mo tools if API alternatives are cheaper.

Direct Quotes
"Cursor felt like highway robbery lol."
"I'm a one man team but right now just Claude code."
"Save a lot of money compared to cursor and I get a lot more usage."

Interview 2: The "Burn-Aware" CTO
Role: CTO of a 12-person AI startup
Current Stack: GitHub Copilot (Team), Claude Pro (Personal), ChatGPT (Team)

Key Findings
Shadow AI: Tribal loyalty to different models causes tool duplication across departments.

Operational Waste: Paying for "seat ghosts" (former employees) because auditing takes too much time.

Surprising Insight: The cost of manual auditing is perceived as higher than the $80/mo waste.

Direct Quotes
"We definitely have 'seat ghosts.' I know I'm paying for at least three people who aren't even on the Slack anymore..."
"The devs want Cursor, the marketing team wants ChatGPT Plus... It's a mess."
"If you showed me I could save $1,000 across the year with one click, I’d authorize that immediately."

Interview 3: The "Comfort-First" Freelancer
Role: Senior Frontend Freelancer
Current Stack: Cursor (Pro), ChatGPT Plus

Key Findings
The "UI Tax": Views the $20/mo subscription as a necessary "productivity tax" for visual comfort.

Mobile Utility: Keeps ChatGPT Plus primarily for non-coding features like Voice Mode and mobile access.

Switching Friction: Locked in by custom GPTs and established workflows.

Direct Quotes
"I know I could save money using the API, but I’m a visual person. I’ll pay the 'Cursor tax'..."
"I keep ChatGPT Plus specifically for the voice mode while I'm driving."
"Unless you can migrate my 50+ Custom GPTs... I’m probably not moving."
